# ROS2 Autonomous Navigation and Obstacle Avoidance Rover System

## Overview
This projsct is an implementation of a multi-node ROS2 system using both Python and C++.

## Features
- Sensor node (Python)
- Controller node (Python)
- Motor node (C++)
- Uses geometry_msgs/Twist for velocity control

## Architecture
Sensor --> Controller --> Motor
