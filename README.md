# ROS2 Autonomous Navigation and Obstacle Avoidance Rover System

## Overview
This project is an implementation of a multi-node ROS2 system using both Python and C++.

## Features
- Sensor node (Python)
- Controller node (Python)
- Motor node (C++)
- Uses geometry_msgs/Twist for velocity control
- Has multi directional sensing and obstacle avoidance logic

## Architecture
Sensor --> Controller --> Motor
